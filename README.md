# Engenharia de Prompt e Aplicações em IA

Este repositório reúne estudos, práticas e exemplos relacionados à **engenharia de prompt** e ao uso de **Inteligência Artificial** em diferentes contextos.  
O objetivo é explorar como prompts bem estruturados podem melhorar a interação com modelos de linguagem e potencializar aplicações práticas.

## 📌 Conteúdo
- Conceitos básicos de engenharia de prompt  
- Exemplos de prompts aplicados em diferentes cenários  
- Casos de uso em aplicações reais de IA  
- Boas práticas e estratégias para criação de prompts eficazes  

## 🚀 Objetivo
Fornecer uma base de conhecimento e experimentação para quem deseja aprender, aplicar e evoluir na área de **IA generativa** e **engenharia de prompt**.

## 🔧 Tecnologias
- Modelos de linguagem (LLMs)  
- Ferramentas de IA generativa  
- Aplicações práticas em diferentes domínios  

## 🤝 Contribuições
Contribuições são bem-vindas!

## 🧠 Atividade: Estrutura de Prompt (4 Zonas)

### 🔹 Prompt Original
Crie um post sobre o calendário acadêmico de provas do UDF.

### 🔹 Prompt Estruturado

**Papel:** Você é um embaixador do UDF.  

**Tarefa:** Crie um post sobre o calendário acadêmico de provas do UDF.  

**Contexto:** O conteúdo é destinado a alunos ingressantes, que precisam entender melhor os períodos de avaliação e organização acadêmica.  

**Formato:** Escreva um texto com no máximo 280 caracteres, utilizando um tom acolhedor, claro e motivador.

**Objetivo:** Demonstrar a criação de um prompt claro e eficiente  

**Prompt:**  
Explique o que é inteligência artificial de forma simples, como se estivesse falando com alguém que nunca teve contato com tecnologia.
